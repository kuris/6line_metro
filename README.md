# 서울 지하철 6호선 — 탈출 어드벤처

> MUD 스타일 텍스트 어드벤처 + 지하철 ASCII 아트 상시 패널

---

## 📁 파일 구조

```
index.html                  // HTML 골격, JS 로드 순서 정의
css/
  style.css                 // 다크 터미널 테마, 2컬럼 + 모바일 반응형
js/
  engine.js                 // 출력(print/seq), 사운드(sfx), UI 엔진, 상태(G)
  train.js                  // 지하철 ASCII 아트 상시 패널 (TrainPanel)
  stations.js               // 38개 역 데이터 (한자명·유래 포함)
  station_trivia.js         // 이벤트 없는 역 통과 정보 카드 (한자·유래·주말코스 등)
  scenes/
    intro.js                // 인트로 → 이름 입력 → 출발역 선택 → 상행/하행 방향 선택
    station_events.js       // 역 간 이동 엔진 + 14개 이벤트 씬
    ending.js               // 엔딩, 등급 판정, 방향·구간별 통계
  main.js                   // 진입점, 모바일 탭 전환, 미니맵
```

## ✅ 완료된 기능 (최신)

### 🕐 시간대 시스템 (NEW)
- **5가지 시간대** 선택: 새벽첫차 / 출근러시 / 한낮 / 퇴근러시 / 심야막차
- 시간대별 객차 분위기·혼잡도·분위기 묘사 다름
- stat-bar에 현재 시간대 상시 표시
- **랜덤 소사건** 시간대 가중치 풀 (새벽전용/출근전용/심야전용 포함)
- 엔딩 분기에 시간대 반영 (새벽영웅·심야영혼 특수엔딩)

### 🎲 랜덤 소사건 시스템 (NEW)
- **20가지 소사건** 풀 (공통 + 시간대 전용)
- 통과 역에서 30% 확률 발생 / 이벤트 역 이동 중 20% 추가 발동
- 선택지가 있는 소사건 (점수·미션 연동)
- 연속 같은 사건 방지 로직

### 🏁 다중 엔딩 6종 (NEW)
| 엔딩 | 조건 | 특징 |
|------|------|------|
| 🏅 서울 시민 영웅 | 30점+ | 일반 최고 |
| ⭐ 따뜻한 이웃 | 18점+ | 일반 좋음 |
| 🚇 평범한 하루 | 8점+ | 일반 보통 |
| 🌑 지하철 유령 | 8점 미만 | 일반 낮음 |
| 🌙 새벽의 수호자 | 새벽+20점+ | **특수** |
| 🌃 심야의 영혼 | 심야+미션4+ | **특수** |
| 🔮 수수께끼의 승객 | 메모지+테이프+15점+ | **비밀** |

### 📤 공유하기 (NEW)
- 엔딩 화면 📤 공유하기 버튼
- QR코드 (게임 URL) + 텍스트 복사
- 트위터 공유 / 카카오 클립보드 복사

### 💾 QR 세이브 시스템 (NEW)
- 헤더 💾 버튼 → QR코드 + URL 문자열 동시 표시
- 모바일: QR 스캔 → `?save=` URL → 즉시 게임 재개
- PC: URL 복사 → 주소창 붙여넣기

### 핵심 게임 흐름
- **인트로 → 이름 입력** : 승객 이름 설정 (엔터 또는 확인 버튼)
- **출발역 선택** : 5개 그룹(서부·마포·용산·동부·북동부)에서 주요 역 선택
  - 한자명 병기 (예: 응암 · 鷹岩) 표시
  - ★ = 이벤트 역 표시
- **상행/하행 방향 선택** (NEW)
  - ▲ 상행: 선택역 → 봉화산 방향 (index 증가)
  - ▼ 하행: 선택역 → 응암 방향 (index 감소)
  - 출발역에 따라 가능한 방향만 표시 (응암=상행만, 봉화산=하행만)
  - 방향 선택 전 역명 한자 유래 ASCII 카드 표시
- **역 이동** : 방향에 따라 +1 또는 -1 인덱스 이동
- **이벤트 역** : 14개 역에서 도덕적 선택지 이벤트 (점수·미션 시스템)
- **통과 역** : 17개 비이벤트 역에서 trivia 카드 표시
- **엔딩** : 방향·구간·점수 통계, 등급 판정

### 역명 한자 유래 (NEW)
- **38개 역 전원** : `hanja` (한자명) + `hanjaDesc` (유래 한 줄 요약) 필드 추가
- **여정 시작 시** : 출발역 한자 유래 ASCII 박스 표시
- **통과 역 trivia 카드** : 한자 타입 카드에서 깊은 설명 제공
- **역 헤더** : 이동 중 역명 옆 한자명 병기

### 이벤트 없는 역 통과 정보 (trivia)
- **17개 역** 전용 trivia 데이터:
  - `hanja` : 한자 유래 상세 설명 (획 분석 포함)
  - `origin` : 지명 유래 이야기
  - `story` : 동네 이야기·역사
  - `weekend` : 주말에 가면 좋은 곳 추천
  - `fact` : 잡학 상식
- 통과 시 한자 유래 ASCII 박스 + 추가 카드 텍스트 표시

### 지하철 ASCII 상시 패널
- idle / arriving / boarding / running / crowded / event / danger / ending 8가지 상태
- 역 도착 시 애니메이션, 노선 스트립, 탑승 로그

### 모바일 지원
- 3탭 UI (스토리 / 열차뷰 / 로그)
- 이벤트 발생 시 탭 뱃지 알림
- 100dvh, iOS input zoom 방지, 44px 터치 타겟

## 🗺 게임 흐름 요약

```
인트로 → 이름 입력
  → 출발역 선택 (5그룹, 한자명 표시)
    → 상행▲ / 하행▼ 방향 선택 + 역명 유래 카드
      → 여정 시작 (출발역 이벤트 or 이동)
        → 이벤트 역: 선택지 → 점수/미션
        → 통과 역: 한자 유래 + trivia 카드 → 계속 이동
      → 종착역 도착 → 엔딩 (방향·구간별 통계 + 등급)
```
